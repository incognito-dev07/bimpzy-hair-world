## Bimpzy Hair World - Hair Styling & Wig E-commerce Platform

A full-featured e-commerce platform for hair styling services and wig products, built with Node.js, Express, and SQL.js.

## Features
- **Product Management** - Browse and purchase wigs and hair products
- **Service Booking** - Schedule appointments for hair services
- **Shopping Cart** - Add products to cart and checkout via WhatsApp
- **Admin Dashboard** - Manage products, services, and bookings
- **Mobile Responsive** - Fully responsive design for all devices
- **Secure Authentication** - Admin authentication with bcrypt hashing

## Tech Stack

- **Stack**: JavaScript, CSS, Node.js, Express.js, SQL.js
- **Security**: bcryptjs, express-validator, helmet, rate-limiting
